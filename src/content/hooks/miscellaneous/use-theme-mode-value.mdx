## Import

```js
import { useThemeModeValue } from "@prismane/core";
```

The `useThemeModeValue` hook accepts the following parameters:

| Name    | Type  | Description                                | Default Value |
| ------- | ----- | ------------------------------------------ | ------------- |
| `dark`  | _any_ | The respective value for dark theme mode.  | -             |
| `light` | _any_ | The respective value for light theme mode. | -             |

## Return Value

The `useThemeModeValue` hook returns the following variable:

| Name    | Type  | Description               |
| ------- | ----- | ------------------------- |
| `value` | _any_ | The current active value. |

## Usage

```jsx preview
function Demo() {
  const { toggleThemeMode } = usePrismaneTheme();

  const background = useThemeModeValue(["base", 300], ["base", 900]);

  return (
    <Center w="100%" h={fr(72)} bg={background}>
      <Button onClick={toggleThemeMode}>Toggle Theme Mode</Button>
    </Center>
  );
}
```
