## Import

```js
import { useId } from "@prismane/core/hooks";
```

## Return Value

The `useId` hook returns the following variable:

| Name | Type     | Description       |
| ---- | -------- | ----------------- |
| `id` | _string_ | The generated id. |

## Usage

```jsx preview
function Demo() {
  const id = useId();

  return <Text>Refresh to regenerate id: {id}</Text>;
}
```
