## Import

```js
import { useNetworkStatus } from "@prismane/core/hooks";
```

## Return Value

The `useNetworkStatus` hook returns the following variable:

| Name       | Type      | Description                                              |
| ---------- | --------- | -------------------------------------------------------- |
| `isOnline` | _boolean_ | The boolean that shows if the user is online or offline. |

## Usage

To test this hook, you can try turning on or off your internet connection.

```jsx preview
function Demo() {
  const online = useNetworkStatus();

  return (
    <Stack>
      <Text>You network is: {online ? "online" : "offline"}</Text>
    </Stack>
  );
}
```
