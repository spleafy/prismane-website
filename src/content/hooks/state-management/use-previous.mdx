## Import

```js
import { usePrevious } from "@prismane/core/hooks";
```

## Parameters

The `usePrevious` hook accepts the following parameters:

| Name    | Type  | Description                       | Default Value |
| ------- | ----- | --------------------------------- | ------------- |
| `value` | _any_ | The value that should be tracked. | -             |

## Return Value

The `usePrevious` hook returns the following variable:

| Name       | Type  | Description                                |
| ---------- | ----- | ------------------------------------------ |
| `previous` | _any_ | The previous value of the passed variable. |

## Usage

```jsx preview
function Demo() {
  const [count, setCount] = useState(0);

  const previous = usePrevious(count);

  return (
    <Stack>
      <Text>Current: {count}</Text>
      <Text>Previous Value: {previous}</Text>
      <Button onClick={() => setCount(count + 1)}>Increment</Button>
    </Stack>
  );
}
```
