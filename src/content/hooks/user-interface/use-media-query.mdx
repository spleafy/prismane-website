---
title: 'useMediaQuery'
description: 'useMediaQuery hook provides a simple way to handle media queries.'
---

## Import

```js
import { useMediaQuery } from '@prismane/core/hooks';
```

## Parameters

The `useMediaQuery` hook accepts the following parameters:

| Name    | Type     | Description                             | Default Value |
| ------- | -------- | --------------------------------------- | ------------- |
| `query` | _string_ | The media query that should be tracked. | -             |

## Return Value

The `useMediaQuery` hook returns the following variable:

| Name      | Type      | Description                                                         |
| --------- | --------- | ------------------------------------------------------------------- |
| `matches` | _boolean_ | The boolean that shows if the current media query is true or false. |

## Usage

```jsx preview
function Demo() {
  const isMobile = useMediaQuery('(max-width: 768px)');

  return (
    <Text
      cl={(theme) => (theme.mode === 'dark' ? ['base', 200] : ['base', 700])}
    >
      Resize screen to change <br />
      Current View: {isMobile ? 'Mobile' : 'Desktop'}
    </Text>
  );
}
```
