---
title: 'useOutsideClick'
description: "useOutsideClick hook provides a simple way to handle a click out of the element's constraints."
---

## Import

```js
import { useOutsideClick } from '@prismane/core/hooks';
```

## Parameters

The `useOutsideClick` hook accepts the following parameters:

| Name       | Type              | Description                                             | Default Value |
| ---------- | ----------------- | ------------------------------------------------------- | ------------- |
| `ref`      | _RefObject\<any>_ | The ref of the element that will be tracked.            | -             |
| `callback` | _() => void_      | The function that will be called when the event occurs. | -             |

## Usage

```jsx preview
function Demo() {
  const ref = useRef(null);

  const [shown, setShown] = useState(true);

  useOutsideClick(ref, () => setShown(false));

  return (
    <Stack>
      {shown && <Card ref={ref}>Click outside of the card to hide it</Card>}
    </Stack>
  );
}
```
