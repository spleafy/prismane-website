## Import

```js
import { useOutsideClick } from "@prismane/core/hooks";
```

## Usage

```jsx preview
function Demo() {
  const ref = useRef(null);

  const [shown, setShown] = useState(true);

  useOutsideClick(ref, () => setShown(false));

  return (
    <>
      <Button onClick={() => setShown(true)}>Show Paper</Button>

      {shown && <Paper ref={ref}>Click outside to close</Paper>}
    </>
  );
}
```

## API

### Parameters

| Name       | Type             | Description                                             | Default |
| ---------- | ---------------- | ------------------------------------------------------- | ------- |
| `ref`      | `RefObject<any>` | The ref of the element that will be tracked.            | -       |
| `callback` | `() => void`     | The function that will be called when the event occurs. | -       |
