# Paper API

Learn about the Paper component's props and CSS classes.

## Import

```js
import { PaperProps } from "@prismane/core";
```

## Props

`Paper` extends [Flex](/docs/components-api/flex-api) component's props.

| Name     | Type      | Description                                | Default |
| -------- | --------- | ------------------------------------------ | ------- |
| `shadow` | `boolean` | Triggers the open shadow of the container. | false   |

## CSS

`Paper` extends [Flex](/docs/components-api/flex-api) component's CSS classes.

| Rule   | Global Class          | Description                           |
| ------ | --------------------- | ------------------------------------- |
| `root` | `.PrismanePaper-root` | The container of the paper component. |
