## Import

```js
import { ColorFieldProps } from "@prismane/core";
```

## Props

`ColorField` extends [SelectField](/docs/components-api/inputs/select-field-api) component's props.

| Name      | Type                                                                               | Description               | Default |
| --------- | ---------------------------------------------------------------------------------- | ------------------------- | ------- |
| `options` | [PrismaneColors](/docs/getting-started/globals#prismanecolors-type)[] / `string[]` | The options of the field. | -       |

## CSS

`ColorField` extends [Box](/docs/components-api/layout/box-api) component's CSS classes.

| Rule   | Global Class               | Description                                 |
| ------ | -------------------------- | ------------------------------------------- |
| `root` | `.PrismaneColorField-root` | The container of the color field component. |
