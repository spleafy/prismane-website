## Import

```js
import { AutocompleteFieldProps } from "@prismane/core";
```

## Props

`AutocompleteField` extends [SelectField](/docs/components-api/inputs/select-field-api) component's props.

| Name     | Type                       | Description                       | Default |
| -------- | -------------------------- | --------------------------------- | ------- |
| `filter` | `(value, item) => boolean` | The filter function of the field. | -       |

## CSS

`AutocompleteField` extends [Box](/docs/components-api/layout/box-api) component's CSS classes.

| Rule   | Global Class                      | Description                                        |
| ------ | --------------------------------- | -------------------------------------------------- |
| `root` | `.PrismaneAutocompleteField-root` | The container of the autocomplete field component. |
