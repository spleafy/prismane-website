## Import

```js
import { FormProps } from "@prismane/core";
```

## Props

`Form` extends [Box](/docs/components-api/box-api) component's props.

| Name       | Type         | Description                  | Default |
| ---------- | ------------ | ---------------------------- | ------- |
| `onSubmit` | `() => void` | The callback on form submit. | -       |
| `onReset`  | `() => void` | The callback on form reset.  | -       |

## CSS

`Form` extends [Box](/docs/components-api/box-api) component's CSS classes.

| Rule   | Global Class         | Description                          |
| ------ | -------------------- | ------------------------------------ |
| `root` | `.PrismaneForm-root` | The container of the form component. |
