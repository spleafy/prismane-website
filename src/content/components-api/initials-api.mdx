# Initials API

Learn about the Initials component's props and CSS classes.

## Import

```js
import { InitialsProps } from "@prismane/core";
```

## Props

| Name   | Type     | Description                                   | Default |
| ------ | -------- | --------------------------------------------- | ------- |
| `name` | `string` | The name whose initials have to be displayed. | -       |
