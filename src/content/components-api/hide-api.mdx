## Import

```js
import { HideProps } from "@prismane/core";
```

## Props

`Hide` extends [Box](/docs/components-api/box-api) component's props.

| Name         | Type                                                                                     | Description                                           | Default |
| ------------ | ---------------------------------------------------------------------------------------- | ----------------------------------------------------- | ------- |
| `breakpoint` | [PrismaneBreakpoints](/docs/getting-started/globals#prismanebreakpoints-type) / `string` | The breakpoint over which the content will be hidden. | base    |

## CSS

`Hide` extends [Box](/docs/components-api/box-api) component's CSS classes.

| Rule   | Global Class         | Description                          |
| ------ | -------------------- | ------------------------------------ |
| `root` | `.PrismaneHide-root` | The container of the hide component. |
