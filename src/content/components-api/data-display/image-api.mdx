## Import

```js
import { ImageProps } from "@prismane/core";
```

## Props

`Image` extends [Square](/docs/components-api/layout/square-api) component's props.

| Name     | Type                                         | Description                                   | Default |
| -------- | -------------------------------------------- | --------------------------------------------- | ------- |
| `src`    | `string`                                     | The source of the image.                      | -       |
| `srcSet` | `string`                                     | The srcSet property of the image element.     | -       |
| `alt`    | `string`                                     | The alt text of the image.                    | -       |
| `sizes`  | `string`                                     | The sizes property of the image element.      | -       |
| `fit`    | `contain / cover / fill / none / scale-down` | The object CSS property of the image element. | -       |

## CSS

`Image` extends [Square](/docs/components-api/layout/square-api) component's CSS classes.

| Rule   | Global Class          | Description                           |
| ------ | --------------------- | ------------------------------------- |
| `root` | `.PrismaneImage-root` | The container of the image component. |
