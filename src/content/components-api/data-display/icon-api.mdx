## Import

```js
import { IconProps } from "@prismane/core";
```

## Props

`Icon` extends [Square](/docs/components-api/layout/square-api) component's props.

| Name   | Type                                                                                              | Description           | Default |
| ------ | ------------------------------------------------------------------------------------------------- | --------------------- | ------- |
| `size` | [PrismaneBreakpoints](/docs/getting-started/globals#prismanebreakpoints-type) / `string / number` | The size of the icon. | base    |

## CSS

`Icon` extends [Square](/docs/components-api/layout/square-api) component's CSS classes.

| Rule   | Global Class                | Description                                             |
| ------ | --------------------------- | ------------------------------------------------------- |
| `root` | `.PrismaneIcon-root`        | The container of the icon component.                    |
| `root` | `.PrismaneIcon-root-{size}` | The container of the icon component based on it's size. |
