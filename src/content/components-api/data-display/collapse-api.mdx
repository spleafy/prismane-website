## Import

```js
import { CollapseProps } from "@prismane/core";
```

## Props

`Collapse` extends [Animation](/docs/components-api/animation-api) component's props.

| Name   | Type      | Description                                    | Default |
| ------ | --------- | ---------------------------------------------- | ------- |
| `open` | `boolean` | Triggers the state of the collapsed container. | false   |

## CSS

`Collapse` extends [Animation](/docs/components-api/animation-api) component's CSS classes.

| Rule   | Global Class                  | Description                                                      |
| ------ | ----------------------------- | ---------------------------------------------------------------- |
| `root` | `.PrismaneCollapse-root`      | The container of the collapse container.                         |
| `root` | `.PrismaneCollapse-root-open` | The container of the collapse container based on the open state. |
