## Import

```js
import { SpinnerProps } from "@prismane/core";
```

## Props

`Spinner` extends [Flex](/docs/components-api/layout/flex-api) component's props.

| Name   | Type                                                                          | Description                        | Default |
| ------ | ----------------------------------------------------------------------------- | ---------------------------------- | ------- |
| `size` | [PrismaneBreakpoints](/docs/getting-started/globals#prismanebreakpoints-type) | The size of the spinner component. | base    |

## CSS

`Spinner` extends [Flex](/docs/components-api/layout/flex-api) component's CSS classes.

| Rule   | Global Class                   | Description                                                |
| ------ | ------------------------------ | ---------------------------------------------------------- |
| `root` | `.PrismaneSpinner-root`        | The container of the spinner component.                    |
| `root` | `.PrismaneSpinner-root-{size}` | The container of the spinner component based on it's size. |
