## Import

```js
import {
  DrawerProps,
  DrawerHeaderProps,
  DrawerFooterProps,
} from "@prismane/core";
```

## Props

### Drawer

`Drawer` extends [Paper](/docs/components-api/paper-api) and [Animation](/docs/components-api/animation-api) component's props.

| Name       | Type                                                                                              | Description                                           | Default  |
| ---------- | ------------------------------------------------------------------------------------------------- | ----------------------------------------------------- | -------- |
| `open`     | `boolean`                                                                                         | Triggers the open state of the dialog.                | false    |
| `closable` | `boolean`                                                                                         | Triggers the closable state of the dialog.            | -        |
| `onClose`  | `() => void`                                                                                      | Callback that will be executed on closing the dialog. | () => {} |
| `size`     | [PrismaneBreakpoints](/docs/getting-started/globals#prismanebreakpoints-type) / `string / number` | The size of the drawer.                               | base     |
| `position` | `right / left / top / bottom`                                                                     | The position of the drawer.                           | left     |

## CSS

### Drawer

`Drawer` extends [Paper](/docs/components-api/paper-api) and [Animation](/docs/components-api/animation-api) components' CSS classes.

| Rule   | Global Class                | Description                                            |
| ------ | --------------------------- | ------------------------------------------------------ |
| `root` | `.PrismaneDrawer-root`      | The wrapper box of the drawer.                         |
| `root` | `.PrismaneDrawer-root-open` | The wrapper box of the drawer based on the open state. |

### Drawer.Header

`Drawer.Header` extends [Flex](/docs/components-api/flex-api) component's CSS classes.

| Rule   | Global Class                 | Description                                       |

| `root` | `.PrismaneDrawerHeader-root` | The wrapper box of the drawer's header component. |
