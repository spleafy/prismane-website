# Card API

Learn about the Card component's props and CSS classes.

## Import

```js
import { CardProps, CardHeaderProps, CardFooterProps } from "@prismane/core";
```

## Props

### Card

`Card` extends [Paper](/docs/components-api/paper-api) component's props.

| Name     | Type      | Description                 | Default |
| -------- | --------- | --------------------------- | ------- |
| `shadow` | `boolean` | Triggers the card's shadow. | -       |

### Card.Header

`Card.Header` extends [Flex](/docs/components-api/flex-api) component's props.

### Card.Footer

`Card.Footer` extends [Flex](/docs/components-api/flex-api) component's props.

## CSS

### Card

`Card` extends [Paper](/docs/components-api/paper-api) component's CSS classes.

| Rule   | Global Class         | Description                  |
| ------ | -------------------- | ---------------------------- |
| `root` | `.PrismaneCard-root` | The wrapper box of the card. |

### Card.Header

`Card.Header` extends [Flex](/docs/components-api/flex-api) component's CSS classes.

| Rule   | Global Class               | Description                                     |
| ------ | -------------------------- | ----------------------------------------------- |
| `root` | `.PrismaneCardHeader-root` | The wrapper box of the card's header component. |

### Card.Footer

`Card.Footer` extends [Flex](/docs/components-api/flex-api) component's CSS classes.

| Rule   | Global Class               | Description                                     |
| ------ | -------------------------- | ----------------------------------------------- |
| `root` | `.PrismaneCardFooter-root` | The wrapper box of the card's footer component. |
