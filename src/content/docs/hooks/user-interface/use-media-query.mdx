## Overview
The useMediaQuery hook provides a straightforward way to respond to CSS media query changes in your React components. It accepts a media query string as input and returns a boolean value indicating whether the query currently matches the user's device characteristics. This hook is essential for building responsive interfaces that adapt to different screen sizes, device capabilities, or user preferences, allowing you to conditionally render components or apply different styles based on the current viewport or device properties.

## Import

```js
import { useMediaQuery } from '@prismane/core/hooks';
```

## Parameters

The `useMediaQuery` hook accepts the following parameters:

| Name    | Type     | Description                             | Default Value |
| ------- | -------- | --------------------------------------- | ------------- |
| `query` | _string_ | The media query that should be tracked. | -             |

## Return Value

The `useMediaQuery` hook returns the following variable:

| Name      | Type      | Description                                                         |
| --------- | --------- | ------------------------------------------------------------------- |
| `matches` | _boolean_ | The boolean that shows if the current media query is true or false. |

## Usage

```jsx preview
function Demo() {
  const isMobile = useMediaQuery('(max-width: 768px)');

  return (
    <Text
      cl={(theme) => (theme.mode === 'dark' ? ['base', 200] : ['base', 700])}
    >
      Resize screen to change <br />
      Current View: {isMobile ? 'Mobile' : 'Desktop'}
    </Text>
  );
}
```
