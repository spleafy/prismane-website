## Import

```js
import { ShowProps } from '@prismane/core';
```

## Props

`Show` extends [Box](/docs/components-api/box-api) component's props.

| Name         | Type                                                                                     | Description                                          | Default |
| ------------ | ---------------------------------------------------------------------------------------- | ---------------------------------------------------- | ------- |
| `breakpoint` | [PrismaneBreakpoints](/docs/getting-started/globals#prismanebreakpoints-type) / `string` | The breakpoint over which the content will be shown. | base    |

## CSS

`Show` extends [Box](/docs/components-api/box-api) component's CSS classes.

| Rule   | Global Class         | Description                          |
| ------ | -------------------- | ------------------------------------ |
| `root` | `.PrismaneShow-root` | The container of the show component. |
