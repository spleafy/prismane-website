## Import

```js
import {
  AccordionProps,
  AccordionControlProps,
  AccordionIconProps,
  AccordionItemProps,
  AccordionPanelProps
} from '@prismane/core';
```

## Props

### Accordion

`Accordion` extends [Flex](/docs/components-api/flex-api) component's props.

| Name           | Type     | Description                         | Default |
| -------------- | -------- | ----------------------------------- | ------- |
| `defaultValue` | `string` | The default value of the accordion. | -       |

### Accordion.Control

`Accordion.Control` extends [Flex](/docs/components-api/flex-api) and [Transition](/docs/components-api/transition-api) components' props.

### Accordion.Icon

`Accordion.Icon` extends [Flex](/docs/components-api/flex-api) component's props.

### Accordion.Item

`Accordion.Item` extends [Flex](/docs/components-api/flex-api) component's props.

| Name    | Type            | Description                      | Default |
| ------- | --------------- | -------------------------------- | ------- |
| `value` | `string / null` | The value of the accordion item. | -       |

### Accordion.Panel

`Accordion.Panel` extends [Flex](/docs/components-api/flex-api) and [Animation](/docs/components-api/animation-api) components' props.

## CSS

### Accordion

`Accordion` extends [Flex](/docs/components-api/flex-api) component's CSS classes.

| Rule   | Global Class              | Description                       |
| ------ | ------------------------- | --------------------------------- |
| `root` | `.PrismaneAccordion-root` | The wrapper box of the accordion. |

### Accordion.Control

`Accordion.Control` extends [Flex](/docs/components-api/flex-api) and [Transition](/docs/components-api/transition-api) components' CSS classes.

| Rule   | Global Class                            | Description                                                        |
| ------ | --------------------------------------- | ------------------------------------------------------------------ |
| `root` | `.PrismaneAccordionControl-root`        | The wrapper box of the accordion control.                          |
| `root` | `.PrismaneAccordionControl-root-active` | The wrapper box of the accordion control if the control is active. |

### Accordion.Icon

`Accordion.Icon` extends [Flex](/docs/components-api/flex-api) component's CSS classes.

| Rule   | Global Class                         | Description                                                  |
| ------ | ------------------------------------ | ------------------------------------------------------------ |
| `root` | `.PrismaneAccordionIcon-root`        | The wrapper box of the accordion icon.                       |
| `root` | `.PrismaneAccordionIcon-root-active` | The wrapper box of the accordion icon if the icon is active. |

### Accordion.Item

`Accordion.Item` extends [Flex](/docs/components-api/flex-api) component's CSS classes.

| Rule   | Global Class                  | Description                            |
| ------ | ----------------------------- | -------------------------------------- |
| `root` | `.PrismaneAccordionItem-root` | The wrapper box of the accordion item. |

### Accordion.Panel

`Accordion.Panel` extends [Flex](/docs/components-api/flex-api) and [Animation](/docs/components-api/animation-api) components' CSS classes.

| Rule   | Global Class                          | Description                                                    |
| ------ | ------------------------------------- | -------------------------------------------------------------- |
| `root` | `.PrismaneAccordionPanel-root`        | The wrapper box of the accordion panel.                        |
| `root` | `.PrismaneAccordionPanel-root-active` | The wrapper box of the accordion panel if the panel is active. |
