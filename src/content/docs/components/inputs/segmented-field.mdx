## Import

```js
import { SegmentedField } from '@prismane/core';
```

## Usage

The `SegmentedField` is based on the [`Field`](/docs/components/inputs/field) component and extends it's respective props.

This page does not include all of the features that are supported by the field. Visit the [`Field`](/docs/components/inputs/field) documentation page to learn about all of the features.

Use the _options_ prop to add options to the field. The structure of each option must be as follows:

- _value_: The value of the option that will be set as the field's value if chosen.
- _element_: The element that will be displayed as an item inside the field.

```jsx preview
function Demo() {
  const [value, setValue] = useState('star');

  return (
    <SegmentedField
      label="Choose a shape:"
      value={value}
      onChange={(e) => setValue(e.target.value)}
      options={[
        { element: 'Star', value: 'star' },
        { element: 'Circle', value: 'circle' },
        { element: 'Square', value: 'square' }
      ]}
    />
  );
}
```

### Read Only Field

Like the [`Field`](/docs/components/field) component, you can make the field read only by passing the _readOnly_ prop. If the _readOnly_ prop is set, the field will not show suggestions and won't change.

```jsx preview
function Demo() {
  const [value, setValue] = useState('star');

  return (
    <SegmentedField
      label="Choose a shape:"
      value={value}
      onChange={(e) => setValue(e.target.value)}
      options={[
        { element: 'Star', value: 'star' },
        { element: 'Circle', value: 'circle' },
        { element: 'Square', value: 'square' }
      ]}
      readOnly
    />
  );
}
```

### Disabled Field

Like the [`Field`](/docs/components/field) component, you can disable the field by passing the _disabled_ prop. If the _disabled_ prop is set, the user would not be able to interact with the field.

```jsx preview
function Demo() {
  const [value, setValue] = useState('star');

  return (
    <SegmentedField
      label="Choose a shape:"
      value={value}
      onChange={(e) => setValue(e.target.value)}
      options={[
        { element: 'Star', value: 'star' },
        { element: 'Circle', value: 'circle' },
        { element: 'Square', value: 'square' }
      ]}
      disabled
    />
  );
}
```

### useForm Example

The `SegmentedField` component, like all other field components of Prismane, has a great integration with our [`useForm`](/docs/hooks/use-form) hook.

Firstly, import the `useForm` hook like so:

```js
import { useForm } from '@prismane/core/hooks';
```

Then do the following:

```jsx preview
function Demo() {
  const { register } = useForm({
    fields: {
      hook_segmented: {
        value: 'star'
      }
    }
  });

  return (
    <SegmentedField
      label="Controlled SegmentedField"
      placeholder="Controlled Field"
      options={[
        { element: 'Star', value: 'star' },
        { element: 'Circle', value: 'circle' },
        { element: 'Square', value: 'square' }
      ]}
      {...register('hook_segmented')}
    />
  );
}
```

### Field Ref

Use the _ref_ prop to get the ref of the input element.

```jsx
import { useRef } from 'react';
import { SegmentedField } from '@prismane/core';

function Demo() {
  const ref = useRef(null);

  return <SegmentedField ref={ref} />;
}
```

### Accessibility

If you want to use the `SegmentedField` without passing a value to the _label_ prop, you will have to add it an _area-label_ so that the input will be accessible by screen readers.

```jsx
import { SegmentedField } from '@prismane/core';

function Demo() {
  return <SegmentedField aria-label="SegmentedField's Input" />;
}
```

If you use the _label_ prop, the field will be accessible by screen readers.

```jsx
import { SegmentedField } from '@prismane/core';

function Demo() {
  return <SegmentedField label="SegmentedField's Input" />;
}
```
