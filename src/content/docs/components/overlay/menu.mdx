## Import

Prismane's `Menu` component has 3 additional inner components:

- _Menu.Item_: The container wrapper of each menu item.
- _Menu.Label_: A label container of the menu.
- _Menu.Icon_: An icon that is displayed inside the menu item.

```js
import { Menu } from '@prismane/core';
```

## Usage

```jsx preview
function Demo() {
  const [open, setOpen] = useState(false);

  return (
    <Stack>
      <Button onClick={() => setOpen(!open)}>Toggle Menu</Button>
      <Menu w={fr(64)} open={open}>
        <Menu.Item>
          <Menu.Icon>
            <User />
          </Menu.Icon>
          Account
        </Menu.Item>
        <Menu.Item>
          <Menu.Icon>
            <Chat />
          </Menu.Icon>
          Messages
        </Menu.Item>
        <Menu.Item>
          <Menu.Icon>
            <GearSix />
          </Menu.Icon>
          Settings
        </Menu.Item>
      </Menu>
    </Stack>
  );
}
```

### Labeled Menu

Use the `Menu.Label` component to separate the menu into different labeled subparts.

It provides the needed spacing and adapts to the theme mode changes.

```jsx preview
function Demo() {
  const [open, setOpen] = useState(false);

  return (
    <Stack w={fr(64)}>
      <Button onClick={() => setOpen(!open)}>Toggle Menu</Button>
      <Menu w={fr(64)} open={open}>
        <Menu.Label>Account</Menu.Label>
        <Menu.Item>Account</Menu.Item>
        <Menu.Item>Messages</Menu.Item>
        <Menu.Item>Settings</Menu.Item>

        <Menu.Label>Workspace</Menu.Label>
        <Menu.Item>Overview</Menu.Item>
        <Menu.Item>Members</Menu.Item>
        <Menu.Item>Workspace Settings</Menu.Item>
      </Menu>
    </Stack>
  );
}
```

### Menu Colors

Use the _color_ prop of the `Menu.Item` component to change it's background and text colors. Values can be all colors from the [Prismane Color Palette](/docs/styling/color-palette) or _primary_ and _base_ colors from the theme.

```jsx preview
function Demo() {
  const [open, setOpen] = useState(false);

  return (
    <Stack w={fr(64)}>
      <Button onClick={() => setOpen(!open)}>Toggle Menu</Button>
      <Menu w={fr(64)} open={open}>
        <Menu.Label>Account</Menu.Label>
        <Menu.Item>
          <Menu.Icon>
            <User />
          </Menu.Icon>
          Account
        </Menu.Item>
        <Menu.Item>
          <Menu.Icon>
            <Chat />
          </Menu.Icon>
          Messages
        </Menu.Item>
        <Menu.Item>
          <Menu.Icon>
            <GearSix />
          </Menu.Icon>
          Settings
        </Menu.Item>
        <Menu.Item color="red">
          <Menu.Icon>
            <SignOut />
          </Menu.Icon>
          Log Out
        </Menu.Item>
      </Menu>
    </Stack>
  );
}
```
