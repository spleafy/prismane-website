## Import

Prismane's `Popover` component has 2 additional inner components:

- _Popover.Control_: The container that wraps the control that will trigger the popover.
- _Popover.Panel_: The container that wraps the content of the popover.

```js
import { Popover } from '@prismane/core';
```

## Usage

```jsx preview
<Popover>
  <Popover.Control>
    <Button>Toggle</Button>
  </Popover.Control>
  <Popover.Panel>
    <Text
      cl={(theme) => (theme.mode === 'dark' ? ['base', 200] : ['base', 700])}
    >
      JavaScript was developed in 10 days by Brendan Eich.
    </Text>
  </Popover.Panel>
</Popover>
```

### Popover Positions

You can change the position of the popover by using the _position_ prop. You can choose any of [PrismanePositions](/docs/getting-started/globals#prismanepositions) as a value. By default, the _bottom_ position is used.

```jsx preview
function Demo() {
  const [position, setPosition] = useState('bottom');

  const [point, setPoint] = useState('middle');

  return (
    <Stack gap={fr(4)} align="center">
      <Radio.Group
        value={position}
        onChange={(e) => setPosition(e.target.value)}
      >
        <Radio label="Top" value="top" />
        <Radio label="Bottom" value="bottom" />
        <Radio label="Left" value="left" />
        <Radio label="Right" value="right" />
      </Radio.Group>
      <Radio.Group value={point} onChange={(e) => setPoint(e.target.value)}>
        <Radio label="Start" value="start" />
        <Radio label="None (e.g. right, left)" value="none" />
        <Radio label="End" value="end" />
      </Radio.Group>
      <Popover position={point !== 'none' ? `${position}-${point}` : position}>
        <Popover.Control>
          <Button>Toggle</Button>
        </Popover.Control>
        <Popover.Panel>
          <Text
            cl={(theme) =>
              theme.mode === 'dark' ? ['base', 200] : ['base', 700]
            }
          >
            This is the content of the popover!
          </Text>
        </Popover.Panel>
      </Popover>
    </Stack>
  );
}
```
