## Import

```js
import { less } from "@prismane/core/validators";
```

## Parameters

The `less` validator accepts the following parameters:

| Name        | Type                  | Description                       |
| ----------- | --------------------- | --------------------------------- |
| `value`     | _number_              | The value that will be validated. |
| `number`    | _number_              | The number that will be compared. |
| `fieldName` | _string_, _undefined_ | The name of the field.            |

## Usage

The `less` validator tests the provided value and checks if it is less the provided number.

```ts
const value = 30;

const number = 20;

const error = less(value, number);

console.log(error); // This number must be less than 20!
```

If there is no error, _null_ is returned.

```ts
const value = 10;

const number = 20;

const error = less(value, number);

console.log(error); // null
```

### Custom Error Message

Pass the _fieldName_ parameter to the validator, which is the third parameter, to customize the error message to make it more personalized to the given field.

```ts
const value = 90;

const number = 70;

const error = less(value, number, "Age");

console.log(error); // Age must be less than 70!
```

### useForm Example

The `less` validator, like all other validators of Prismane, has a great integration with our [`useForm`](/docs/hooks/use-form) hook.

Firstly, import the `useForm` hook like so:

```js
import { useForm } from "@prismane/core/hooks";
```

Then do the following:

```jsx preview
function Demo() {
  const { handleSubmit, register } = useForm({
    fields: {
      number: {
        value: 0,
        validators: {
          less: (v: string) => less(v, 5),
        },
      },
    },
  });

  return (
    <Form
      onSubmit={(e: any) => {
        handleSubmit(e, (v: any) => console.log(v, ""));
      }}
    >
      <NumberField
        placeholder="Enter your number:"
        label="Number:"
        {...register("number")}
      />
      <Button type="submit">Submit</Button>
    </Form>
  );
}
```
