## Styling Components with Prismane

In this section, we will demonstrate how to style components using Prismane. We will cover various styling options, CSS variables, and component-based theming.

### Shorthand Props

Prismane's shorthand props make it easy to quickly style components without writing lengthy CSS rules. This can save you time and effort, and make your code more readable and maintainable. Additionally, Prismane's shorthand props have support for pseudo-classes, at rules and much more.

Instead of lengthy CSS rules, use the shorthand props to effortlessly customize components like the Box without the need for additional CSS.

```css name="index.css" icon={<File />}
.PrismaneBox-root {
  padding: 16px !important;
  background: red;
  color: white;
}
```

We can write the following code, which will produce the same output:

```tsx
<Box p={16} bg="red" cl="white" />
```

The `p` prop sets the padding of the button to `16px`

The `bg` prop sets the background of the button to `red`

The `cl` prop sets the color property of the button to `white`

### Customizing A Button Component

Using the shorthand props that Prismane exposes, you can craft a new component, tailor the Button component to your preferences, and reuse it by following these steps:

```tsx name="NewButton.tsx" icon={<File />}
import { Button, ButtonProps } from '@prismane/core';

const NewButton = ({ children, ...props }: ButtonProps) => {
  return (
    <Button px={18} py={12} br={0} {...props}>
      {children}
    </Button>
  );
};

export default NewButton;
```

The `px` prop sets the horizontal padding of the button to `18px`

The `py` prop sets the vertical padding of the button to `12px`

The `br` prop sets the border-radius property of the button to `0`

<Note>
  ðŸ’¡ We are using the `ButtonProps` type from Prismaneâ€™s `Button` component.
  This way we are ensuring that the new component still has TypeScript support.
</Note>

Following these steps, you have created a fully customized `NewButton` component that still has all of the functionality of the `Button` component.

### Pseudo Classes

All of Prismaneâ€™s shorthand props have support for pseudo-classes. To fully understand all of Prismaneâ€™s shorthand propsâ€™ features, check out this [tutorial](https://www.prismane.io/docs/styling/style-props).

To add a pseudo-class value, we need to pass an array to the shorthand prop. The first value of the array is the normal value, the second value of the array is an object whose keys are the names of the pseudo-classes, and itâ€™s values are the respected values when the pseudo-class is active.

To easily understand how they work, letâ€™s change the background of our button and change it once again on hover. This can be done as follows:

```tsx name="NewButton.tsx" icon={<File />}

  return (
    <Button
      px={18}
      py={12}
      br={0}
      bg={['#40e495', { hover: '#2bb673' }]}
      {...props}
    >
      {children}
    </Button>
  );
};

export default NewButton;
```
