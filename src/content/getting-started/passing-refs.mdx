All the components in Prismane use the _forwardRef_ function, so that you can easily pass a ref to the component by using the `ref` prop.

Here is an example of passing a custom ref to the [`Flex`](/docs/components/layout/flex) component:

```tsx
import { useRef } from 'react';
import { Flex } from '@prismane/core';

const Page = () => {
  const ref = useRef(null);

  return <Flex ref={ref}>Hello, world</Flex>;
};

export default Page;
```
