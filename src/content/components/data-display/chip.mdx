## Import

```js
import { Chip } from "@prismane/core";
```

## Usage

```jsx preview
<Chip>Default</Chip>
```

### Chip Sizes

Use the _size_ prop to change the size of the chip. You can set the value to _xs_, _sm_, _base_, _md_, _lg_. By default, the _base_ value is used.

```jsx preview
<>
  <Chip size="xs">Size xs</Chip>
  <Chip size="sm">Size sm</Chip>
  <Chip size="base">Size base</Chip>
  <Chip size="md">Size md</Chip>
  <Chip size="lg">Size lg</Chip>
</>
```

### Chip Colors

Use the _color_ prop to change the color scheme of the chip. You can use all colors from the [Prismane Color Palette](/docs/styling/color-palette) or _primary_, _base_ colors from the theme.

By default, the _primary_ color is used.

```jsx preview
<>
  <Chip color="red">Chip</Chip>
  <Chip color="orange">Chip</Chip>
  <Chip color="lime">Chip</Chip>
  <Chip color="cyan">Chip</Chip>
  <Chip color="amethyst">Chip</Chip>
</>
```

### Chip With Icon

You can add an icon to the chip, by using the _icon_ prop.

```jsx preview
<Chip icon={<At />}>martinpetrov404</Chip>
```

### Chip Icon Position

To change the icon position of the button, you can use the _iconPosition_ prop. You can set the value to _left_ or _right_. By default, the _left_ value is used.

```jsx preview
<>
  <Chip iconPosition="left" icon={<Lightning />} color="lime">
    Speed
  </Chip>
  <Chip iconPosition="right" icon={<PaperPlaneTilt />} color="diamond">
    Send
  </Chip>
</>
```

### Change Text Color

You can use the _cl_ prop to change the text color of the chip. Values can be all colors from the [Prismane Color Palette](/docs/styling/color-palette), _primary_, _base_ colors from the theme or any valid color.

```jsx preview
<Chip color="base" cl="primary">
  Chip
</Chip>
```
