## Import

```js
import { Tabs } from "@prismane/core";
```

## Usage

```jsx preview
<Tabs defaultValue="first">
  <Tabs.List>
    <Tabs.Tab value="first">First</Tabs.Tab>
    <Tabs.Tab value="second">Second</Tabs.Tab>
    <Tabs.Tab value="third">Third</Tabs.Tab>
  </Tabs.List>

  <Tabs.Panel value="first">First</Tabs.Panel>
  <Tabs.Panel value="second">Second</Tabs.Panel>
  <Tabs.Panel value="third">Third</Tabs.Panel>
</Tabs>
```

### Tabs Variants

Use the _variant_ prop to change the visual appearance of the tabs. You can set the value to _underlined_, _filled_. By default, the _underlined_ variant is used.

```jsx preview
<Tabs defaultValue="first" variant="filled">
  <Tabs.List>
    <Tabs.Tab value="first">First</Tabs.Tab>
    <Tabs.Tab value="second">Second</Tabs.Tab>
    <Tabs.Tab value="third">Third</Tabs.Tab>
  </Tabs.List>

  <Tabs.Panel value="first">First</Tabs.Panel>
  <Tabs.Panel value="second">Second</Tabs.Panel>
  <Tabs.Panel value="third">Third</Tabs.Panel>
</Tabs>
```

### Tabs Alignment

Pass the _justify_ prop to change the alignment of our tabs list. Use a value of _end_ to show the list in the end.

Supported values are _start_, _center_ and _end_. By default, the _start_ value is used.

```jsx preview
<Tabs defaultValue="first">
  <Tabs.List justify="end">
    <Tabs.Tab value="first">First</Tabs.Tab>
    <Tabs.Tab value="second">Second</Tabs.Tab>
    <Tabs.Tab value="third">Third</Tabs.Tab>
  </Tabs.List>

  <Tabs.Panel value="first">First</Tabs.Panel>
  <Tabs.Panel value="second">Second</Tabs.Panel>
  <Tabs.Panel value="third">Third</Tabs.Panel>
</Tabs>
```
