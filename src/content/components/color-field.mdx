## Import

```js
import { ColorField } from "@prismane/core";
```

## Usage

```jsx
function Demo() {
  const [color, setColor] = useState("");

  return (
    <ColorField
      placeholder="No color chosen"
      label="Choose a color:"
      value={color}
      onChange={(e) => setColor(e.target.value)}
    />
  );
}
```

### Custom Set Of Colors

`ColorField` accepts a parameter of `options`, in which you can specify what color the field should offer as options. They can either be [PrismaneColors](/docs/getting-started/globals#prismanecolors-type) or custom colors.

```jsx
function Demo() {
  const [color, setColor] = useState("");

  return (
    <ColorField
      options={["teal", "#1e1e1e"]}
      placeholder="No color chosen"
      label="Custom color:"
      value={color}
      onChange={(e) => setColor(e.target.value)}
    />
  );
}
```
