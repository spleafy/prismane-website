# NumberField

NumberField displays a field that handles numbers, allowing incrementing and decrementing the value.

## Import

```js
import { NumberField } from "@prismane/core";
```

## Usage

```jsx
function Demo() {
  const [value, setValue] = useState(0);

  return (
    <NumberField value={value} onChange={(e) => setValue(e.target.value)} />
  );
}
```

### With Icon

```jsx
function Demo() {
  const [value, setValue] = useState(0);

    <NumberField
      value={value}
      onChange={(e) => setValue(e.target.value)}

}
```
