## Import

```js
import { Footer } from '@prismane/core';
```

## Usage

The `Footer` component renders a `footer` element. It is an essential part of an application and should be used to maintain the Semantic HTML that an app or website should have.

```jsx
<Footer>Renders a footer element.</Footer>
```
