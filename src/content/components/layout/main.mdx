## Import

```js
import { Main } from "@prismane/core";
```

## Usage

The `Main` component renders a `main` element. It is an essential part of an application and should be used to maintain the Semantic HTML that an app or website should have.

```jsx
<Main>Renders a main element.</Main>
```
