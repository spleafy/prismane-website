## Import

```js
import { Header } from "@prismane/core";
```

## Usage

The `Header` component renders a `header` element. It is an essential part of an application and should be used to maintain the Semantic HTML that an app or website should have.

```jsx
<Header>Renders a footer element.</Header>
```
