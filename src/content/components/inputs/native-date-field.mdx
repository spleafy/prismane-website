## Import

```js
import { NativeDateField } from "@prismane/core";
```

## Usage

The `NativeDateField` is based on the [`Field`](/docs/components/inputs/field) component and extends it's respective props.

This page does not include all of the features that are supported by the field. Visit the [`Field`](/docs/components/inputs/field) documentation page to learn about all of the features.

```jsx preview
<NativeDateField name="date" />
```

### Controlled

```jsx
function Demo() {
  const [date, setDate] = useState("");

  return (
    <NativeDateField
      name="date"
      value={date}
      onChange={(e) => setDate(e.target.value)}
    />
  );
}
```

### Read Only Field

Like the [`Field`](/docs/components/field) component, you can make the field read only by passing the _readOnly_ prop. If the _readOnly_ prop is set, the field will not show options and won't change.

```jsx preview
function Demo() {
  const [date, setDate] = useState("");

  return (
    <NativeDateField
      placeholder="No date chosen"
      label="Choose a date:"
      value={date}
      onChange={(e) => setDate(e.target.value)}
      readOnly
    />
  );
}
```

### Disabled Field

Like the [`Field`](/docs/components/field) component, you can disable the field by passing the _disabled_ prop. If the _disabled_ prop is set, the user would not be able to interact with the field.

```jsx preview
function Demo() {
  const [date, setDate] = useState("");

  return (
    <NativeDateField
      placeholder="No date chosen"
      label="Choose a date:"
      value={date}
      onChange={(e) => setDate(e.target.value)}
      disabled
    />
  );
}
```

### useForm Example

The `NativeDateField` component, like all other field components of Prismane, has a great integration with our [`useForm`](/docs/hooks/use-form) hook.

Firstly, import the `useForm` hook like so:

```js
import { useForm } from "@prismane/core/hooks";
```

Then do the following:

```jsx preview
function Demo() {
  const { register } = useForm({
    fields: {
      hook_native_date: {
        value: "",
      },
    },
  });

  return (
    <NativeDateField
      label="Controlled NativeDateField"
      placeholder="Controlled Field"
      {...register("hook_native_date")}
    />
  );
}
```

### Field Ref

Use the _ref_ prop to get the ref of the input element.

```jsx
import { useRef } from "react";
import { NativeDateField } from "@prismane/core";

function Demo() {
  const ref = useRef(null);

  return <NativeDateField ref={ref} />;
}
```

### Accessibility

If you want to use the `NativeDateField` without passing a value to the _label_ prop, you will have to add it an _area-label_ so that the input will be accessible by screen readers.

```jsx
import { NativeDateField } from "@prismane/core";

function Demo() {
  return <NativeDateField aria-label="NativeDateField's Input" />;
}
```

If you use the _label_ prop, the field will be accessible by screen readers.

```jsx
import { NativeDateField } from "@prismane/core";

function Demo() {
  return <NativeDateField label="NativeDateField's Input" />;
}
```
