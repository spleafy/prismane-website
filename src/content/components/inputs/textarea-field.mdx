## Import

```js
import { TextareaField } from "@prismane/core";
```

## Usage

The `TextareaField` is based on the [`Field`](/docs/components/inputs/field) component and extends it's respective props.

This page does not include all of the features that are supported by the field. Visit the [`Field`](/docs/components/inputs/field) documentation page to learn about all of the features.

```jsx preview
function Demo() {
  const [value, setValue] = useState("");

  return (
    <TextareaField
      value={value}
      onChange={(e) => setValue(e.target.value)}
      label="Text:"
      placeholder="Enter Text"
    />
  );
}
```

### Customize Size

Use the _w_ and _h_ props to change the size of the field.

```jsx preview
function Demo() {
  const [value, setValue] = useState("");

  return (
    <TextareaField
      w={fr(64)}
      h={fr(96)}
      value={value}
      onChange={(e) => setValue(e.target.value)}
      label="Resized Textarea:"
      placeholder="Enter Text"
    />
  );
}
```

### Read Only Field

Like the [`Field`](/docs/components/field) component, you can make the field read only by passing the _readOnly_ prop. If the _readOnly_ prop is set, the field won't change.

```jsx preview
function Demo() {
  const [value, setValue] = useState("I am read only!");

  return (
    <TextareaField
      label="Text:"
      placeholder="Enter Text"
      value={value}
      onChange={(e) => setValue(e.target.value)}
      readOnly
    />
  );
}
```

### Disabled Field

Like the [`Field`](/docs/components/field) component, you can disable the field by passing the _disabled_ prop. If the _disabled_ prop is set, the user would not be able to interact with the field.

```jsx preview
function Demo() {
  const [value, setValue] = useState("I am disabled!");

  return (
    <TextareaField
      label="Text:"
      placeholder="Enter Text"
      value={value}
      onChange={(e) => setValue(e.target.value)}
      disabled
    />
  );
}
```

### useForm Example

The `TextareaField` component, like all other field components of Prismane, has a great integration with our [`useForm`](/docs/hooks/use-form) hook.

Firstly, import the `useForm` hook like so:

```js
import { useForm } from "@prismane/core/hooks";
```

Then do the following:

```jsx preview
function Demo() {
  const { register } = useForm({
    fields: {
      hook_textarea: {
        value: "",
      },
    },
  });

  return (
    <TextareaField
      label="Controlled TextareaField"
      placeholder="Controlled Field"
      {...register("hook_textarea")}
    />
  );
}
```

### Field Ref

Use the _ref_ prop to get the ref of the input element.

```jsx
import { useRef } from "react";
import { TextareaField } from "@prismane/core";

function Demo() {
  const ref = useRef(null);

  return <TextareaField ref={ref} />;
}
```

### Accessibility

If you want to use the `TextareaField` without passing a value to the _label_ prop, you will have to add it an _area-label_ so that the input will be accessible by screen readers.

```jsx
import { TextareaField } from "@prismane/core";

function Demo() {
  return <TextareaField aria-label="TextareaField's Input" />;
}
```

If you use the _label_ prop, the field will be accessible by screen readers.

```jsx
import { TextareaField } from "@prismane/core";

function Demo() {
  return <TextareaField label="TextareaField's Input" />;
}
```
