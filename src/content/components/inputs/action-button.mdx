## Import

```js
import { ActionButton } from "@prismane/core";
```

## Usage

The `ActionButton` component uses the [`Button`](/docs/components/button) component, but only renders an icon. Hence, you have to provide a _aria-label_ to the button, so that screen readers can give the button a meaning.

```jsx preview
<ActionButton icon={<MagnifyingGlass />} aria-label="Search Components" />
```

### ActionButton Variants

Like the [`Button`](/docs/components/button) component, you can change the button's visual appearance by using the _variant_ prop.

```jsx preview
<>
  <ActionButton
    variant="primary"
    icon={<ThumbsUp />}
    aria-label="Like Prismane's Post"
  />
  <ActionButton
    variant="secondary"
    icon={<ThumbsUp />}
    aria-label="Like Prismane's Post"
  />
  <ActionButton
    variant="tertiary"
    icon={<ThumbsUp />}
    aria-label="Like Prismane's Post"
  />
  <ActionButton
    variant="text"
    icon={<ThumbsUp />}
    aria-label="Like Prismane's Post"
  />
</>
```

### ActionButton Sizes

Like the [`Button`](/docs/components/button) component, you can change the size of the button using the _size_ prop.

```jsx preview
<>
  <ActionButton
    size="xs"
    icon={<DownloadSimple />}
    aria-label="Download Prismane's Source Code"
  />
  <ActionButton
    size="sm"
    icon={<DownloadSimple />}
    aria-label="Download Prismane's Source Code"
  />
  <ActionButton
    size="base"
    icon={<DownloadSimple />}
    aria-label="Download Prismane's Source Code"
  />
  <ActionButton
    size="md"
    icon={<DownloadSimple />}
    aria-label="Download Prismane's Source Code"
  />
  <ActionButton
    size="lg"
    icon={<DownloadSimple />}
    aria-label="Download Prismane's Source Code"
  />
</>
```

### ActionButton Color

Like the [`Button`](/docs/components/button) component, you can change the color of the button using the _color_ prop.

```jsx preview
<ActionButton color="base" icon={<GearSix />} aria-label="Open Settings" />
```

### Custom Button

The `ActionButton` component uses the [`Button`](/docs/components/button) component and inherits all of its props, which means that we can easily customize the button only using props.

```jsx preview
<ActionButton
  icon={<Play />}
  aria-label="Play Soundgarden - Black Hole Sun"
  br="full"
  variant="secondary"
  color="lime"
/>
```
