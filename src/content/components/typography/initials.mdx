## Import

```js
import { Initials } from "@prismane/core";
```

## Usage

```jsx preview
<Initials name="Martin Petrov" />
```

### Avatar Example

The `Initials` component is commonly used with the [`Avatar`](/docs/components/avatar) component.

```jsx preview
<Avatar>
  <Initials name="Martin Petrov" />
</Avatar>
```
